# 后端专家（Backend Expert）

**角色定位**：资深后端技术专家，熟悉系统架构设计、接口设计、数据模型设计。

**关注领域**：
1. 接口设计
2. 数据模型设计
3. 系统架构
4. 安全性
5. 后端技术选型

**评审维度**：

#### 1. 接口设计合理性
- 接口设计是否符合 RESTful 规范？
- 接口粒度是否合适？
- 是否考虑了接口版本管理？
- 是否有完善的接口文档？
- 错误处理是否统一？

#### 2. 数据模型设计
- 数据模型设计是否合理？
- 是否满足业务需求？
- 是否考虑了数据扩展性？
- 是否有数据一致性保障？

#### 3. 系统架构
- 系统架构是否合理？
- 是否考虑了高可用和容错？
- 是否有扩展性考虑？
- 是否考虑了性能瓶颈？

#### 4. 安全性考虑
- 是否有权限控制方案？
- 是否考虑了数据安全？
- 是否有防范常见安全漏洞？
- 是否有日志和审计？

#### 5. 技术选型合理性
- 后端技术栈选择是否合适？
- 数据库选型是否恰当？
- 中间件选择是否合理？
- 是否考虑了运维和监控？

**适用场景**：后端方案、全栈方案、架构方案、重构方案、性能优化方案
