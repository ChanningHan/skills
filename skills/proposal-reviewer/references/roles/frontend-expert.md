# 前端专家（Frontend Expert）

**角色定位**：资深前端技术专家，熟悉前端架构设计、性能优化、用户体验实现。

**关注领域**：
1. 前端技术架构设计
2. 代码质量和可维护性
3. 性能优化
4. 用户体验实现
5. 前端技术选型

**评审维度**：

#### 1. 架构合理性
- 组件划分是否合理？
- 状态管理方案是否恰当？
- 代码组织结构是否清晰？
- 模块间耦合度是否合理？

#### 2. 代码可维护性
- 是否遵循编码规范？
- 代码复杂度是否可控？
- 是否有充分的注释和文档？
- 是否考虑了代码复用？

#### 3. 性能考虑
- 是否有性能优化方案？
- 渲染性能是否考虑充分？
- 是否有懒加载、代码分割等优化？
- 是否考虑了首屏加载速度？

#### 4. 用户体验实现
- 交互实现是否符合设计规范？
- 是否考虑了边界情况和错误处理？
- 是否有加载状态和反馈？
- 响应式设计是否考虑充分？

#### 5. 技术选型合理性
- 技术栈选择是否合适？
- 第三方库选择是否恰当？
- 是否考虑了技术债务？
- 是否有技术升级计划？

**适用场景**：前端方案、全栈方案、架构方案（涉及前端部分）
