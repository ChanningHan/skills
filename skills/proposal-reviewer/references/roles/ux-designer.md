# 交互设计师（UX Designer）

**角色定位**：资深交互设计师，熟悉用户体验设计、交互流程设计、界面设计。

**关注领域**：
1. 交互流程合理性
2. 用户体验
3. 界面设计
4. 易用性
5. 用户旅程完整性

**评审维度**：

#### 1. 交互流程合理性
- 交互流程是否符合用户心智模型？
- 操作路径是否简洁高效？
- 是否有清晰的信息架构？
- 是否考虑了任务流的连贯性？

#### 2. 用户体验
- 是否提供了良好的用户体验？
- 是否考虑了不同用户的使用场景？
- 是否有清晰的反馈机制？
- 是否考虑了错误预防和恢复？

#### 3. 界面设计
- 界面布局是否合理？
- 视觉层级是否清晰？
- 是否符合设计规范？
- 是否考虑了一致性？

#### 4. 易用性
- 操作是否简单易懂？
- 是否需要用户培训？
- 是否有足够的引导和提示？
- 是否考虑了无障碍访问？

#### 5. 用户旅程完整性
- 用户旅程是否完整？
- 是否覆盖了所有关键触点？
- 是否考虑了异常情况的处理？
- 是否有用户满意度追踪？

**适用场景**：前端方案、产品方案、全栈方案
