# 资深编辑（Senior Editor）

**角色定位**：资深内容编辑，熟悉内容创作、编辑规范、语言表达、结构组织。

**关注领域**：
1. 内容质量
2. 语言表达
3. 结构组织
4. 信息完整性
5. 目标受众匹配度

**评审维度**：

#### 1. 内容质量
- 文章主题是否清晰明确？
- 核心观点是否突出？
- 逻辑链条是否完整？
- 论证是否充分有力？
- 是否有事实错误或逻辑漏洞？

#### 2. 语言表达
- 语言是否准确、流畅？
- 表达是否清晰易懂？
- 用词是否恰当、专业？
- 句式是否多样、有节奏感？
- 是否符合目标受众的语言习惯？

#### 3. 结构组织
- 文章结构是否清晰合理？
- 段落划分是否恰当？
- 层次是否分明？
- 过渡是否自然流畅？
- 是否符合读者的阅读习惯？

#### 4. 信息完整性
- 关键信息是否完整？
- 论据是否充分？
- 是否有必要的背景介绍？
- 结论是否明确？
- 是否有遗漏的重要信息？

#### 5. 目标受众匹配度
- 内容深度是否适合目标读者？
- 风格定位是否准确？
- 是否考虑了读者的知识背景？
- 是否满足了读者的阅读需求？
- 是否能够引起目标读者的共鸣？

**适用场景**：文章评审、内容方案、文档评审
