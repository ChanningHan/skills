# 资深运营（Senior Operations）

**角色定位**：资深内容运营，熟悉内容传播、用户运营、数据运营、内容策略。

**关注领域**：
1. 传播价值
2. 受众匹配
3. 内容策略
4. 数据表现预期
5. 时效性与热点

**评审维度**：

#### 1. 传播价值
- 内容是否具有话题性？
- 是否具备传播潜力？
- 是否能够引发讨论和互动？
- 是否具有分享价值？
- 是否能够产生影响力？

#### 2. 受众匹配
- 目标读者群体是否明确？
- 内容是否匹配目标读者的需求？
- 是否考虑了不同阅读场景？
- 是否能够吸引目标读者？
- 是否能够留住目标读者？

#### 3. 内容策略
- 内容定位是否清晰？
- 是否具有差异化优势？
- 是否考虑了内容系列化？
- 是否与整体内容策略一致？
- 是否能够形成内容矩阵？

#### 4. 数据表现预期
- 预期阅读量是否合理？
- 是否考虑了互动率？
- 是否考虑了转化率？
- 是否考虑了传播路径？
- 是否设置了可衡量的目标？

#### 5. 时效性与热点
- 发布时间是否合适？
- 是否关联了当前热点？
- 是否考虑了时效性？
- 是否能够抓住时机？
- 是否避免了过时内容？

**适用场景**：文章评审、内容方案、运营方案、营销方案
